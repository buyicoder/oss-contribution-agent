# Scripts

This directory is reserved for future automation.

Add scripts only after the corresponding manual workflow has been run enough times to be stable.

Planned scripts:

- `scan_repos`: refresh target repository metadata.
- `scan_issues`: find candidate L1-L3 issues.
- `dedupe_prs`: check assignments, linked PRs, and recent closed PRs.
- `status_report`: summarize open PRs, issue comments, and maintainer replies.

Automation should support judgment, not replace it.
